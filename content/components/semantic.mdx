Several components from the popular [Semantic UI](https://react.semantic-ui.com). 
Please visit this website for the list of options. Contributions are welcome to support
full catalogue of Semantic Components.

Please visit the [official React Semantic UI page](https://react.semantic-ui.com) for comprehensive examples of 
various combination of control properties.

### Installation

```sh
yarn add @toryjs/components-semantic-ui
```

#### Important

Make sure you include the `semantic.css` somewhere in your document, 
whether as an import in your html or in import in your css, for example from the
`semantic-ui-css` package. To include the semantic-ui in your html, add this to your `index.html`.

```html
<link
  href="https://cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.3.2/semantic.min.css"
  rel="stylesheet"
  type="text/css"
/>
```

### Dropdown

Semantic [Dropdown](https://react.semantic-ui.com/modules/dropdown/) component is a very versatile component which can obtain the list of options from a variety of sources:

* `Options` (bound): Explicitly specified list of options. If you are binding the options to a handler, make sure it is not an asynchronous handler.
* `Async Options`: Asynchronous handler, returning `Promise` of `Options`.
* `Schema Source`: From a schema item's enum (Enumerations).

The values in the dropdown can be filtered by a value from the dataset. To specify such cascading filter, you need to define two values:

* `Filter Source`: the name of the dataset field (e.g. `country` with value `es`)
* `Filter Column`: the name of the column from the list of options that will match the dataset field. 
  For example following option list will filter out only `{ text: 'Barcelona', value: 'bcn', country: 'es' }`:

    ```bash
    { text: 'Sydney', value: 'syd', country: 'au' },
    { text: 'Melbourne', value: 'mel', country: 'au' },
    { text: 'Bari', value: 'bari', country: 'it' },
    { text: 'Barcelona', value: 'bcn', country: 'es' },
    { text: 'Rozhanovce', value: 'roz', country: 'sk' }
    ```

  Check the examples below.
