### Installation

```sh
yarn add @toryjs/components-react-router
```

### Router Provider

A react router provider allows you to use react router routes. There **must** be at least one parent react router provider. Plase see examples below for use of ReactRouter component. You can fake the current route in the editor.

<ReactRouterProps name="ReactRouterProvider" />

### Route

A react router route displays the related **page** based on the current url. Pages are defined as separate forms. Examples below use fake routes and as a result are only viewable in the editor.

All route parameters are stored in the root parameters of your dataset. For example route `/catalogue/:productName/:productId/` would save `productName` and `productId` to the root of your dataset.
