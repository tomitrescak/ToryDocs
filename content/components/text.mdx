Set of base components usefull to build your forms or websites. 
Please choose the component of your interest from the right menu.

### Installation

```bash
yarn add @toryjs/components-vanilla
```

```
Documentation in Progress
```

- **Formula** - Dynamic expression evaluating data in your dataset/
- **Image** - a simple `<img />`
- **Link** - a simple `<a href />`. If you want to use cusom links, please use appropriate catalogues, such as the one for React router.
- **Markdown** - renders markdown text
- **Selector** - Modifies the dataset value when clicked
- **Text** - Displays either a static or bound text.
