### Fetch Provider

#### Installation

```bash
yarn add @toryjs/components-vanilla
```

Adds support for performing `fetch` requests to json endpoints. 
You can provide options of the fetch request using the `options` field. For a full list of options, please see the [full documentation of the fetch](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API/Using_Fetch).

You can also *fake* the data returned from the endpoint. This will work only in the edit mode.

### Alert Provider

#### Installation

```bash
yarn add @toryjs/components-react-alert
```

Adds support for React alert. This instance of alert is added to context under `context.alert`, 
which can be used in handlers. Following are the handlers we used to show alerts in the example below.
Please see the [full documentation of the package](https://www.npmjs.com/package/react-alert).

```ts
const handlers = {
  info({ context }) {
    context.providers.alert.info('Info');
  },
  error({ context }) {
    context.providers.alert.error('Error');
  },
  success({ context }) {
    context.providers.alert.success('Success');
  }
};
```
