### Alert Provider

#### Installation

```bash
yarn add @toryjs/components-react-alert
```

Adds support for React alert. This instance of alert is added to context under `context.alert`, which can be used in handlers.


### List of components

- **Auth Item** - Displays content based on authorisation state
- **Context** - Creates fake application context
- **React Alert** - Adds support for react alert
