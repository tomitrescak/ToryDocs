### Auth Item

#### Installation

```bash
yarn add @toryjs/components-vanilla
```

This component shows the child components only if user is authorised to view them. 
You can specify the list of roles that are able to view this content. 
Also, for test purposes, you can *disable* the component. Auth Item component 
works well with fake Context during development, allowing to fake the logged in user. 
*This components assumes that you keep `user` object in the `Context` with property `roles` of type `string[]`*.
